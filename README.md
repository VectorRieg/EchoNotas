# Echo Notas

EchoNota é um sistema web voltado para a gestão de notas escolares, com funcionalidades de cadastro de alunos, professores, e boletins.

# Funcionalidade
<ul>
  <li>Java</li>
  <li>Spring Boot</li>
  <li>Spring MVC</li>
  <li>HTML</li>
  <li>CSS</li>
  <li>MySQL</li>
  <li>Hibernate</li>
  <li>Maven</li>
</ul>
